Workshops


At AppHack, speakers across a variety of disciplines share their knowledge by giving 50 minute seminars. Thank you to all who have agreed to spread their knowledge!

Dr. Cindy Norris :: How fast can you add 1 billion numbers? A hands-on workshop!



Off-the-shelf desktops and laptops combine multi-core CPUs with hundreds of GPU cores bringing tera-scale computing power. Although traditionally used for graphics, the release of CUDA by NVIDIA now allows the GPU to solve problems in any number of disciplines. This hands-on workshop will introduce attendees to parallel programming that harnesses the power of the CPU and the GPU to solve the problem of adding a billion numbers.

Michelle Melton :: Hacking the Amazon Alexa



Michelle is a graduate student in Computer Science and a Web Developer/Programmer with Learning Technology Services here at App State. Her master's thesis is building an Alexa skill for the Moodle learning management system. During this workshop, you will learn to use Node.js and Amazon Web Services to program an Amazon Alexa to respond to specifically created voice commands.

Austin Keener :: Web Development - Easier Than It Seems



When you think WebDev, do you have nightmares about PHP? Does the phrase Client-Server MVC make you panic? This talk is geared towards changing your feelings on WebDev by introducing you to Vue.js, a Javascript Component Framework with functionality that you've always wanted for the web. We will step through the surprisingly simple and straightforward process of developing an entire website using nothing more than Javascript and some minimal HTML.

Tim Ransom :: Emacs - Making Life Easier



A text editor is a programmer's Swiss Army Knife. Unfortunately, like most programs meant for developers, there is an overwhelming number of features and configurations. Tim will show a handful of emacs commands to ease the development workflow and let you focus on the fun parts of code.

Hunter McIntyre :: Getting Started in Unity



As the president of the Video Game Development Club, Hunter has been a hobbyist game developer for 7 years. In his workshop, we can learn the basics of Unity, as we explore the various components that make a Unity game. We will provide a project file for hands on experience, that will allow you to design your own level.

Sebastian Collins & Andrew Thorp :: Shell Scripting with BASH



As the president and vice president of the Linux @ App club, respectively, Sebastian and Andrew have plenty of experience with sharing their knowledge. For their Apphack workshop, they will teach the basics of Bash, a "Unix shell" - a command line interface for interacting with the operating system. A Bash script has the ability to run an entire script of shell commands, allowing for far more complex instructions.

Keith Pohl, T-Metrics :: Amazon Lex Contact Center Bots



Keith Pohl, the VP of Technology at T-Metrics, a world-class provider of Premise-based, Hybrid and Cloud Multi-Channel Contact Centers / Operating Systems. T-Metrics' ability to leverage the existing infrastructure within an organization is a key reason to their success. Keith will be discussing Why and How T-Metrics is using Amazon Lex to power the next generation of contact center applications.

Jessica Kennedy, Inmar :: CSS Fundamentals



Intended for a complete novice to CSS, Jessica will cover the basics of CSS, as well as the best practices to keep you out of CSS hell. Additionally, we will look at some of the most commonly misunderstood/useful bits of CSS that outside the very basics. Finally, the "block model" will be explained in more detail, and we will look into using pseudo elements, finishing with the foundations of writing scaleable code.

Jessica Kennedy, Inmar :: CSS Architecture & Preprocessor Fundamentals



If you've ever asked, "Why is my CSS doing that?!", this is the class for you. We will cover some more in-depth CSS topics, specifically about architecting CSS to make development a breeze. We'll talk about common CSS patterns including BEM, ITCSS & SMACSS and how they can help make CSS simpler to understand. Along with architecture, we'll also cover some of the basics of CSS preprocessors and how developers can get started using them to their advantage.
